# Pagina-Fruteria-2026-Franco-Segovia
> Aplicación web de una tienda de frutas desarrollada con HTML, CSS y JavaScript vanilla.
Proyecto realizado como práctica académica para un parcial | ejercicio de curso en Programacion III, UTN.

> 🚧 Primera versión del proyecto (actualmente no responsive).

## Funcionalidades

- Visualización dinámica de productos
- Filtro por nombre
- Ordenamiento por:
  - A-Z
  - Z-A
  - Menor precio
  - Mayor precio
- Carrito de compras
- Incrementar y disminuir cantidades
- Vaciar carrito
- Persistencia con localStorage
- Contador automático de productos
- Cálculo del precio total

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
